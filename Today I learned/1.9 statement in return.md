# Statement in return 

```py
def is_positive(num):
    if num > 0:
        return True
    else:
        return None
```

We can rewrite this again

```py
def is_positive(num):
    return True if num > 0 else None
```

```py
def is_positive(num):
    return num > 0 if num > 0 else None
```

How do we simplify this return statement?


```py
def is_even(num):
    if num % 2 == 0 :
        return True
    return False
```

<details>
    <summary>Answer</summary>

    ```py
    def is_even(num):
        return num % 2 == 0
    ```

</details>
