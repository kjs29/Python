# Statement in return 

```py
def is_positive(num):
    if num > 0:
        return True
    else:
        return None
```

We can rewrite this again

```py
def is_positive(num):
    return True if num > 0 else None
```

```py
def is_positive(num):
    return num > 0 if num > 0 else None
```
