# Polymorphism

Normally when we create a function, there is only one output that can be created.

```py
def hello():
    print("hello")

hello()   #hello
hello()   #hello
hello()   #hello
```

But Polymorphism allows same named function to create multiple outputs
```py
class Person:
    def say_hello(self):
        print("hello")
        
class Korean(Person):
    def say_hello(self):
        print("안녕하세요?")

class Indian(Person):
    def say_hello(self):
        print("ਸਤ ਸ੍ਰੀ ਅਕਾਲ"

p1 = Person()
p2 = Korean()
p3 = Indian()

party = [p1,p2,p3]
for each_person in party:
    each_person.say_hello()
```
