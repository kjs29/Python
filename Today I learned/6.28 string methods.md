# 6/28 Today I learned...

To check if string is made of capital letters only
```py
"HELLO".isupper()       #True
"Hello".isupper()       #False
```
To check if string is made of only whitespaces

```py
" ".isspace()         #True
"".isspace()          #False
"\n".isspace()        #True
"\t".isspace()        #True
```
